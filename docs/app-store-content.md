## Description (4000 chars max)

Routine Tracker is the class-planning tool built for yoga and Pilates instructors who want to stop staring at Apple Notes during class.

Build your full class structure before you walk in the room — blocks, steps, equipment cues, timing — then run through it seamlessly on your phone while you teach.

**BUILD YOUR CLASS**
• Organize moves into labeled blocks (Warm-Up, Core, Standing Flow, etc.)
• Add steps with names, cues, and emoji tags for quick visual scanning
• Mark high-intensity moves as Pulse to flag them at a glance
• Set equipment per block — ball, band, ring, chair, or none
• Color-code blocks for instant visual rhythm

**QUICK ENTRY — NO TYPING SLOW**
• Quick Add: type exercise names one per line to fill a block instantly
• Quick Build: paste your full class plan using ## headings and --- transitions — the whole routine builds itself in seconds
• Replace your Apple Notes workflow entirely

**RUN YOUR CLASS**
• Clean run view — big, readable, one block at a time
• Quiz Mode to test yourself before class
• Collapse blocks you've already covered
• Landscape support for tablet or stand use

**MANAGE YOUR LIBRARY**
• Save unlimited routines
• Load any saved routine instantly
• Duplicate and remix existing classes
• Mirror blocks (left side / right side) automatically

**YOGA & PILATES READY**
• Switch between Yoga and Pilates discipline modes
• Leveaux studio theme with warm, clean aesthetics
• Works completely offline — no account, no subscription, no sync required
• All data stays on your device

Whether you're teaching mat Pilates, barre, vinyasa, or sculpt — Routine Tracker keeps your plan in your pocket and your focus on your students.

## App Store Screenshots — Required Sizes

Apple requires at least one of:
- **iPhone 6.9"**: 1320 × 2868 px (iPhone 16 Pro Max)
- **iPhone 5.5"**: 1242 × 2208 px (iPhone 8 Plus)

### Suggested Screenshot Sequence (5 max)
1. Main view — a full routine loaded (shows blocks + steps)
2. Edit mode — block sheet open (shows editing UX)
3. Quick Build sheet — typing a class plan
4. Run/quiz mode — clean class view
5. Library — saved routines list

### How to take them
1. Open Xcode Simulator → iPhone 16 Pro Max
2. Run the app, set up a good-looking routine
3. Simulator menu → File → Take Screenshot (or Cmd+S)
4. Screenshots save to Desktop automatically
